# OWASP AppSec Days Portugal 2026

Official website for OWASP AppSec Days Portugal 2026, taking place in Porto, Portugal.

## Overview

This is a static website for the OWASP AppSec Days Portugal 2026 conference. The site features:

- Bilingual support (English/Portuguese)
- Responsive design for all devices
- Program overview with two-day agenda
- Call for Papers (CFP) section with Sessionize integration
- Sponsorship information
- Venue details
- Organizing committee information

## Event Details

- **Date:** 2026
- **Location:** Porto, Portugal
- **Day 1:** Workshops, Trainings, CTF Kick-off
- **Day 2:** Full conference day with keynotes, talks across 2 tracks, and evening social event

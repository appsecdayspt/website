Keynote Speakers

Keynote Speaker

Lukas Weichselbaum

LinkedIn

Senior Staff Security Engineer & Manager, Google Product Security

Lukas Weichselbaum is a Senior Staff Security Engineer and Manager on Google's Product Security team, where for over a decade he has architected scalable defenses that protect billions of users every day. As a champion of secure-by-design, Lukas bootstrapped Google's first AI Agent Security team and leads its Web Security team, both on a mission to eliminate entire classes of vulnerabilities by embedding security primitives across the SDLC and directly into core frameworks and browsers, helping shift the industry from reactive patching to high-assurance, secure-by-default infrastructure.

Beyond Google, Lukas is a driving force in open web standards. As a core contributor to the W3C Web Application Security Working Group, he has co-authored foundational specifications including Content Security Policy Level 3, Trusted Types, and Fetch Metadata.

As of this year, Lukas also leads Google's Agentic Web Security strategy, combining deep web security expertise with advanced AI to prepare the web for a new generation of AI-driven attacks. His team is building agents for large-scale vulnerability discovery and autonomous patching at machine scale, and instilling web secure-by-design principles into AI models themselves, so that the next generation is secure-by-default from the start.
